# Security Policy

The ownCloud team welcomes responsible disclosure of security vulnerabilities and is committed to addressing issues promptly.

## Reporting a Vulnerability

Please report security issues privately via [ownCloud's responsible disclosure process](https://owncloud.com/security/). Do not open a public GitHub issue for security vulnerabilities.

## Security Advisories

Confirmed vulnerabilities are remediated and disclosed in [GitHub Security Advisories](https://github.com/mklos-kw/ocis-github-actions/security/advisories) and in the release notes for the affected version.
